Design a brand-new circle-packing organism from scratch.

1. invent a coherent idea set
2. express that idea set as genes
3. summarize the single defining novelty

Do:
- think in terms of ideas, assumptions, and constructive principles
- make the genes precise enough that they can later be implemented faithfully
- produce one organism with one coherent research direction

Do not:
- jump directly to a finished placement recipe
- write genes that are just vague labels
- rely on parent-like assumptions, because this is a new seed
- treat a hand-tuned arrangement as a substitute for a real idea

## Response format

Return the following sections, each starting with `## SECTION_NAME`.
Do NOT use markdown code fences.

## CORE_GENES
At least 3 bullets.
Each bullet is one concrete idea.

## INTERACTION_NOTES
State what assumptions the idea set makes and where it is likely to help or fail.

## COMPUTE_NOTES
State the computational style of the organism and what kind of procedure it implies.

## CHANGE_DESCRIPTION
Give one unified novelty summary for this organism.
