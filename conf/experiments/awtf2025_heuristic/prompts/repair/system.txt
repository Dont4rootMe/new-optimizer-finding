## Common failure modes to check

1. All parentheses, brackets, and strings are properly closed
2. `solve_case(...)` always returns a string, never `None`
3. The output begins with exactly `N` vertical-wall lines, each of length `N - 1`
4. The output continues with exactly `N - 1` horizontal-wall lines, each of length `N`
5. The output continues with exactly `K` group-id lines, each a single integer in `[0, K - 1]`
6. Every operation line uses one of `g` or `i`, a valid group or robot index, and one of `U`, `D`, `L`, `R`
7. The total operation count satisfies `T <= K * N * N`
8. No file I/O, subprocess, network call, or third-party import is introduced
9. Loops are bounded by an explicit counter or by a budget derived from `K * N * N`
10. The code completes within the configured per-case timeout budget

If the error history contains markdown fence or non-Python preamble warnings, remove any non-Python preamble first.
